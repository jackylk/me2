# 📱 Me2 移动端快速开始指南

## 🚀 5 分钟上手

### 步骤 1: 在手机浏览器中打开 Me2

**选项 A: 使用局域网地址（推荐用于测试）**

1. 确保手机和电脑连接同一个 WiFi
2. 在电脑上运行：
   ```bash
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```
   找到你的局域网 IP（如 192.168.1.100）

3. 在手机浏览器中访问：
   ```
   http://192.168.1.100:3000
   ```

**选项 B: 使用公网地址（生产环境）**

直接访问你部署的域名，如：
```
https://me2.yourdomain.com
```

---

### 步骤 2: 在手机上安装 Me2

#### 📱 iPhone / iPad

1. 打开 **Safari** 浏览器（必须使用 Safari）
2. 访问 Me2 地址
3. 点击底部中间的 **分享按钮** 📤
4. 向下滚动，找到 **"添加到主屏幕"**
5. 点击 **"添加"**
6. 主屏幕会出现 Me2 图标 ✨

**提示**：
- 必须使用 Safari，Chrome/Firefox 不支持
- 如果看不到"添加到主屏幕"，请更新 iOS 系统

#### 🤖 Android

**方法 1: 浏览器自动提示（推荐）**

1. 打开 **Chrome** 或 **Edge** 浏览器
2. 访问 Me2 地址
3. 第 3 次访问时会自动弹出安装提示
4. 点击 **"安装"** 或 **"添加到主屏幕"**

**方法 2: 手动安装**

1. 打开 Chrome/Edge
2. 点击右上角/右下角的 **⋮ 菜单**
3. 选择 **"安装应用"** 或 **"添加到主屏幕"**
4. 确认安装

**提示**：
- 推荐使用 Chrome 或 Edge
- 部分国产浏览器可能不支持 PWA

---

### 步骤 3: 开始使用

1. **点击主屏幕上的 Me2 图标**
   - 会像原生应用一样全屏打开
   - 没有浏览器地址栏

2. **登录或注册**
   - 使用邮箱和密码注册
   - 或使用已有账号登录

3. **开始聊天**
   - 点击底部"聊天"标签
   - 输入消息，体验 AI 陪伴
   - Me2 会记住你说的每件事

4. **探索更多功能**
   - 📝 **记忆**：查看 AI 记住的所有内容
   - 🧠 **画像**：查看 AI 对你的理解
   - 🖼️ **图片**：上传和管理图片记忆
   - 🔔 **关心**：设置主动关心提醒

## 🐛 常见问题

### Q1: 为什么看不到安装按钮？

**可能原因**：
1. 浏览器不支持 PWA
2. 已经安装过
3. 网站不是 HTTPS（除了 localhost）
4. 使用了隐私/无痕模式

**解决方法**：
- iPhone: 使用 Safari 并手动添加到主屏幕
- Android: 使用 Chrome/Edge 并从菜单手动安装

### Q2: 安装后图标是空白的？

**原因**：图标文件未正确加载

**解决方法**：
1. 重新访问网站
2. 删除应用并重新安装
3. 检查图标文件是否存在（/icons/icon-*.png）

### Q3: 离线时能做什么？

**可以**：
- ✅ 打开应用
- ✅ 查看已缓存的对话
- ✅ 浏览记忆列表

**不可以**：
- ❌ 发送新消息
- ❌ 召回新记忆
- ❌ 登录/注册

### Q4: 如何更新应用？

PWA 会自动更新：
- 每次打开检查更新
- 后台自动下载
- 下次启动应用新版本

手动强制更新：
1. 打开应用
2. 下拉刷新
3. 或清除应用数据重新安装

### Q5: 占用多少存储空间？

通常情况：
- 应用本身：5-10 MB
- 缓存数据：10-50 MB
- 总计：< 100 MB（远小于原生应用）

## 💡 使用技巧

### 1. 快速切换功能

使用底部导航栏快速切换：
- 💬 **聊天**：主对话界面
- 🧠 **记忆**：查看所有记忆
- ⋯ **更多**：其他功能菜单

Android 支持长按应用图标显示快捷操作：
- 新对话
- 查看记忆
- （需要在 manifest.json 中配置）

### 3. 多设备同步

在多个设备上安装 Me2：
- 登录同一账号
- 数据自动同步
- 对话记录共享

### 4. 节省流量

安装后启用离线缓存：
- 首屏内容离线可用
- 减少重复加载
- 节省移动数据
